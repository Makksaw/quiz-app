## Installation
1. Clone the repository:
```
git clone https://github.com/Makksaw/quiz-app.git
```

2. Navigate to the project directory:
```
cd quiz-app
```

4. Install the required dependencies:
```
npm install
```

## Run the project:
```
npm start
```

[localhost:3000](https://localhost:3000)
